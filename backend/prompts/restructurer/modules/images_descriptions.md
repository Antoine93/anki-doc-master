# Module: Descriptions des Images

Décris textuellement chaque image en détail.

## Structure attendue

Pour chaque image:
- reference: Référence (ex: "Figure 1")
- detailed_description: Description complète
- key_elements: Éléments clés
- educational_value: Valeur pédagogique

## Format JSON

```json
{"items": [{"reference": "...", "detailed_description": "...", "key_elements": [...], "educational_value": "..."}]}
```
