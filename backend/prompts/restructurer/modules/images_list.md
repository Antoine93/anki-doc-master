# Module: Liste des Images

Liste toutes les images/schémas du document.

## Structure attendue

Pour chaque image:
- description: Description brève
- page: Numéro de page
- type: Type (photo, schéma, diagramme, graphique)

## Format JSON

```json
{"items": [{"description": "...", "page": 1, "type": "schéma"}]}
```
