# Module: Liste des Images

Liste toutes les images, schémas, figures et diagrammes du document pour traitement ultérieur.

## Instructions

- Identifier chaque ressource visuelle significative
- Décrire brièvement ce que représente l'image
- Localiser précisément (section et/ou page)

## Structure attendue

Pour chaque image:
- reference: Référence (ex: "Figure 1", "Schéma 2.3")
- description: Description brève de ce que représente l'image
- section: Section du document où elle apparaît
- page: Numéro de page
- type: Type (photo, schéma, diagramme, graphique, tableau visuel)

## Format JSON

```json
{"items": [{"reference": "Figure 1", "description": "...", "section": "1.2", "page": 5, "type": "schéma"}]}
```
