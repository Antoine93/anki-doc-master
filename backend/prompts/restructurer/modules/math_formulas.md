# Module: Formules Mathématiques

Extrais toutes les formules mathématiques.

## Structure attendue

Pour chaque formule:
- formula: La formule (LaTeX si possible)
- description: Ce qu'elle représente
- variables: Signification des variables
- context: Contexte d'utilisation

## Format JSON

```json
{"items": [{"formula": "...", "description": "...", "variables": {...}, "context": "..."}]}
```
