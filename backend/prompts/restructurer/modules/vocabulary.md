# Module: Vocabulaire

Extrais TOUS les termes techniques, acronymes et concepts spécialisés du document pour constituer un glossaire complet.

## Instructions

- Inclure tous les termes techniques sans exception
- Définitions courtes et précises (1-2 phrases max)
- Si un terme a plusieurs sens, préciser celui qui s'applique au contexte
- Ordre alphabétique recommandé

## Structure attendue

Pour chaque terme:
- term: Le terme ou acronyme
- definition: Définition concise et précise
- context: Section/page où le terme apparaît (optionnel)

## Format JSON

```json
{"items": [{"term": "...", "definition": "...", "context": "..."}]}
```
